# hbridge
## Description
A basic H-Bridge circuit. Usually, they're used for motors. But we don't have those :(. So I just used 2 LEDs. Made by @pizzalover125

## BOM
4x 6mm Push Buttons
1x 2032 Battery Holder
2x 5mm LEDs
2x THT Resistors
1x JLC PCB
